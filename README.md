# Prodigy InfoTech Internship - Task 2

This project is a part of my internship at Prodigy InfoTech. In this task, I worked on customer segmentation using the K-Means clustering algorithm.

## About the Project
The main idea of this project is to group customers based on their annual income and spending score. This helps in understanding different types of customers and their behavior.

## Dataset Used
I used the Mall Customers dataset for this project.  
From the dataset, I selected:
- Annual Income (k$)
- Spending Score (1-100)

## What I Did
- Loaded the dataset using pandas  
- Selected required features  
- Applied K-Means clustering  
- Found clusters of customers  
- Visualized the result using a scatter plot  

## Result
The model divided customers into 5 different clusters.  
Each cluster represents a group with similar spending habits and income levels.

## Tools and Technologies
- Python  
- Pandas  
- Matplotlib  
- Scikit-learn  

## Conclusion
This task helped me understand how clustering works in real-world data and how businesses can use it for customer segmentation.
